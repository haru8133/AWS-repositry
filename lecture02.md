# バージョン管理システム
## どういうもの？
- システムの状態を保存
- コードを修正した後に何か問題が起きた時、前の状態に戻すことができる
- 例）**Git**, SVN

## Git
### 基本的なコマンド
1. リモートリポジトリからローカルリポジトリに複製
　`git clone`
2. ローカルで変更などをして、ローカルリポジトリに変更を反映
  `git add`
  `git commit`
3. リモートリポジトリに変更を反映
  `git push`
4. 他の人が変更したものをローカルリポジトリに反映
  `git pull`
